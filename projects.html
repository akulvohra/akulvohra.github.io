Historic Tours - 4/22/2018 - Made with Unity and C#

Historic Tours was developed at TinoHacks 2018. The goal of the project was to address a lack of interactive history apps on the market. In this application, students can walk through different ancient civilizations in 3D on their computer. Along the way, they will be stopped by markers in which a virtual tour guide will give information about a landmark or an aspect of daily life. This project won for the Top Ten category at TinoHacks.


Solve. - 10/14/2017 - Made with Java Graphics Engine

Solve. is a math puzzle game was developed at Wilhacks 2017. The project is used for elementary school mathematics teachers. Different colored blocks each reperesent a mathematics variable. The objective of the game is to have the same variables on each side. The only functions the students can use are basic mathematics functions (add, subtract, multiply, and divide.)
