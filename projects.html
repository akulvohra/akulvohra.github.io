Projects


Here are a handful of projects I've worked on:


Dyspute

(October 2018 - Present)

Dyspute is a webapp aimed at connecting users to people who have different opinions than themselves with the goal of creating empathy between them.
The project is currently in beta and will be coming out soon at https://dyspute.com


Sentwiment

(Made for HSHacks 2018)

Sentwiment is a natural language processing project I worked on as an introduction to the topic. The model analyzes tweets from Twitters API for polarity and subjectivity and generates a JSON file of the values.
Source code for my project can be found at https://github.com/akulvohra/Sentwiment


Historical Tours

(Made for TinoHacks 2018)

Historical Tours is a fun educational project I made. The idea was to allow students to experience history through a virtual tour. A tour guide would navigate users through a historical city while giving information and fun facts.
Source code for this project can be found at https://github.com/akulvohra/HistoricalTours


Solve.

(Made for WilHacks 2017)

Solve. is a math based puzzle game filled with colorful rectangles. Students are encouraged to apply math operations such as fraction operations, and variables to "solve the puzzle."
The DevPost submission for Solve. can be found at https://devpost.com/software/solve-nxil1y
